Indoor Cat Tracker

BLE-based indoor positioning and session playback

Back to Projects

Overview

An indoor tracking system that estimates a cat’s location using BLE beacons and receivers. The system runs locally and provides a simple web interface to visualize movement over time.

What it does

  • Collects BLE receiver readings for a tracked device
  • Estimates position using beacon/receiver setup and distance approximation
  • Stores time-series location data per tracking session
  • Visualizes sessions with playback controls (slider)

Key Features

  • Session-based tracking (start, end, review past sessions)
  • User-configurable room dimensions and beacon coordinates
  • Heatmap-style visualization and timeline playback
  • Local web dashboard for quick access
